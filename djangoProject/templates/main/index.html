NextGame

The goal of NextGame is to provide a more refined and personal set of game recommendations to gamers.

NextGame uses an aesthetics matching algorithm in combination with a genre tag ranking algorithm to determine game recommendations based off the games you submit.

Instructions

Follow the instructions directly below in order to use NextGame

1. Register

Register an account on Nextgame and then Log in.

Register
2. Submit Games

Search and add some of your favourite games in order for NextGame to determine the best games suited for you.

Add Liked Games
3. Recommendations

You can then go to the "Recommendations" page to view your tailored recommendations that NextGame has chosen for you.
